# Proyecto de "EVENTOS METROPOLITANOS"

>comandos y requerimientos para la instalacion y su ejecucion

_requisitos_
```
php
laravel
livewire
composer
npm
git y git bash
```

### pasos

clonar el repositorio
```
git clone
cd Eventos-Metropolitanos
code .
cp. env.example .env
DB_CONNECTION=mysql
```

ejecutar
```
composer install
npm install
para la ejecucion correcta de css
npm run build
php artisan key:generate
php artisan migrate
```
entrar a tu localhost
